# TaxPilot

AI-powered UAE Tax Compliance Copilot for SMEs.

Goal:
Upload financial documents → analyze tax/compliance risk → explain results in chat → export audit report.

This is an MVP built in 1 week.

Success Criteria:
- Upload works
- Analysis works
- Chat works
- Report exports

Out of Scope:
- Real FTA integration
- Production tax calculation
- Accounting integrations
- Multi-tenancy
- Payments
- Full auth flows

Core User Journey:
Upload
→ Analyze
→ Chat
→ Report

Tech:
Frontend:
- Next.js App Router
- TypeScript
- Tailwind
- Antigravity

Backend:
- Vercel
- Server Actions

Database:
- Supabase

AI:
- Gemini API

Deployment:
- Vercel
